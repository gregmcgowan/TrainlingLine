1.Adding a @Scope annotation on a @Provides ensures that the same instance of the object is used in the
@Component with the same annotation.


2. @Components with a @Scope annotation may only reference un-scoped dependencies or
dependencies with the same scope (apart from @Resuable which can be used in any scoped component).

4. @Reusable is a built in scope that allows you to indicate that the object returned by a
binding may be (but might not be) reused.

It is a "special" scope as adding it will not require a @Scope to be added to the component it is
used in. Also means it can be used in any component regardless of it's scope.
